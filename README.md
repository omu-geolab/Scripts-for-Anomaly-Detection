# Scripts for Anomaly Detection in UAV-Based Crack Inspection

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📌 About / 概要
**[English]** This repository contains the dataset and scripts used in the study:  
*"Improving the Reliability of UAV-Based Crack Inspection of Port Quay Walls Using Anomaly Detection"* (Currently under review).  
The proposed framework introduces a Vision Transformer (ViT)-based anomaly detection module as a post-processing step to effectively reduce false positives (e.g., debris, stains) generated by conventional object detection models during UAV-based concrete infrastructure inspection.

**[日本語]** 本リポジトリは、論文 *"Improving the Reliability of UAV-Based Crack Inspection of Port Quay Walls Using Anomaly Detection"*（現在査読中）で使用されたデータセットおよびスクリプトを公開するものです。
本研究では、UAVを用いたコンクリートインフラ点検において、従来の物体検出モデルで発生しやすい誤検知（ゴミや汚れなど）を効果的に削減するため、Vision Transformer (ViT) をベースとした異常検知モジュールを後処理として導入するフレームワークを提案しています。

---

## 🗂️ Dataset / データセット
**[English]** The dataset used in this repository consists of the following two sources:
1. **SDNET2018 Dataset:** Used for constructing the crack feature center in the anomaly detection module.
2. **UAV Drone Imagery:** Original crack images captured by a general-purpose UAV (drone) at actual port quay walls in this study, used for evaluation and validation.

**[日本語]** 本リポジトリに含まれる（または使用する）データセットの画像は、以下の2つから構成されています。
1. **SDNET2018:** 異常検知モジュールにおける、ひび割れの特徴中心（Feature Center）を構築するために使用。
2. **本研究のドローン空撮画像:** 実際の港湾施設においてドローンで空撮したオリジナルのひび割れ画像であり、手法の評価・検証に使用。

**Reference for SDNET2018 / SDNET2018の引用情報:** > Dorafshan, S.; Thomas, R.J.; Maguire, M. SDNET2018: An Annotated Image Dataset for Non-Contact Concrete Crack Detection Using Deep Convolutional Neural Networks. *Data Brief* 2018, 21, 1664–1668, doi:10.1016/j.dib.2018.11.015.

## 📁 Repository Structure & Script Descriptions / フォルダ構成とスクリプトの説明

**[English]**
* **`trimming.py`**: A script to crop the bounding box regions obtained from the preceding crack detection process. It extracts the corresponding regions from the original image based on the label files containing the bounding box coordinate information.
* **`frangi_filter.py`**: A preprocessing script for applying the Frangi filter. It applies Frangi filtering to each input image and outputs the result.
* **`train_gray.py`**: A training script for the grayscale anomaly detection model. It takes color images as input, internally converts them to grayscale, trains the anomaly detection model, and outputs the trained model.
* **`train_frangi.py`**: A training script for the Frangi filter anomaly detection model. It takes preprocessed images as input, trains the anomaly detection model, and outputs the trained model.
* **`detect_gray.py`**: Performs anomaly detection using grayscale transformation. Utilizing a pre-trained grayscale anomaly detection model, it calculates the anomaly score for each image in the input directory, classifies them based on a threshold, and outputs the results.
* **`detect_frangi.py`**: Performs anomaly detection using the Frangi filter. Utilizing a pre-trained Frangi anomaly detection model, it calculates the anomaly score for each image, classifies them based on a threshold, and outputs the results (both the filtered image and the original image).
* **`check_anormaly.py`**: Outputs the anomaly scores calculated by the anomaly detection model during inference in a heatmap format. Warmer colors indicate lower scores, while cooler colors indicate higher scores.

**[日本語]**
* **`trimming.py`**: 前段のひび割れ検知処理で取得した、バウンディングボックス領域を切り出すスクリプトです。ひび割れ検知処理で取得したバウンディングボックス領域の座標情報が格納されたlabelファイルをもとに、元画像から対応領域を抽出します。
* **`frangi_filter.py`**: frangiフィルタによる前処理用スクリプトです。各入力画像に対し、frangiフィルタによるフィルタリング処理を施して出力します。
* **`train_gray.py`**: グレースケール変換用異常検知モデルの学習用スクリプトです。カラー画像を入力すると、スクリプト内でグレースケール変換が施され、その情報をもとに異常検知モデルが学習され出力されます。
* **`train_frangi.py`**: frangiフィルタ用異常検知モデルの学習用スクリプトです。前処理画像を入力すると、その情報をもとに異常検知モデルが学習され出力されます。
* **`detect_gray.py`**: グレースケール変換での異常検知処理を行うスクリプトです。学習済みのグレースケール変換用異常検知モデルを用いて、入力ディレクトリ内の各画像の異常スコアを算出し、閾値をもとに各画像を分類、出力します。
* **`detect_frangi.py`**: frangiフィルタ適用時の異常検知処理を行うスクリプトです。学習済みのfrangiフィルタ用異常検知モデルを用いて、入力ディレクトリ内の各画像の異常スコアを算出し、閾値をもとに各画像を分類、出力します。出力時は、フィルタリングの画像及び推論元画像がそれぞれ出力されます。
* **`check_anormaly.py`**: 異常検知モデルが推論時に算出した異常スコアをヒートマップ形式で出力するスクリプトです。暖色になるほど低スコアを、寒色になるほど高スコアを示します。
