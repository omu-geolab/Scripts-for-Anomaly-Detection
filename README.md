<u>__check_anormaly.py__</n>
異常検知モデルが推論時に算出した異常スコアをヒートマップ形式で出力するスクリプトです。暖色になるほど低スコアを、寒色になるほど高スコアを示します。

<u>__detect_gray.py__</n>
グレースケール変換での異常検知処理を行うスクリプトです。学習済みのグレースケール変換用異常検知モデルを用いて、入力ディレクトリ内の各画像の異常スコアを算出し、閾値をもとに各画像を分類、出力します。

<u>__detect_frangi.py__</n>
frangiフィルタ適用時の異常検知処理を行うスクリプトです。学習済みのfrangiフィルタ用異常検知モデルを用いて、入力ディレクトリ内の各画像の異常スコアを算出し、閾値をもとに各画像を分類、出力します。出力時は、フィルタリングの画像及び推論元画像がそれぞれ出力されます。

<u>__frangi_filter.py__</n>
frangiフィルタによる前処理用スクリプトです。各入力画像に対し、frangiフィルタによるフィルタリング処理を施して出力します。

<u>__train_gray.py__</n>
グレースケール変換用異常検知モデルの学習用スクリプトです。カラー画像を入力すると、スクリプト内でグレースケール変換が施され、その情報をもとに異常検知モデルが学習され出力されます。

<u>__train_frangi.py__</n>
frangiフィルタ用異常検知モデルの学習用スクリプトです。前処理画像を入力すると、その情報をもとに異常検知モデルが学習され出力されます。

<u>__trimming.py__</n>
前段のひび割れ検知処理で取得した、バウンディングボックス領域を切り出すスクリプトです。ひび割れ検知処理で取得したバウンディングボックス領域の座標情報が格納されたlabelファイルをもとに、元画像から対応領域を抽出します。
